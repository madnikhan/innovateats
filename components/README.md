# Components Directory

This directory contains reusable React components for the Innovat Eats employee management system.

## Structure

- `QRScanner/` - QR code scanner component for employee login/logout
- `Employee/` - Employee-related components (cards, forms, lists)
- `Dashboard/` - Dashboard components for employees and admins
- `Admin/` - Admin-specific components (analytics, employee management)
- `Layout/` - Layout components (navigation, headers, footers)
- `Auth/` - Authentication components
